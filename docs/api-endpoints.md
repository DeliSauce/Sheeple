# API Endpoints

## HTML API

### Root

- `GET /` - loads React web app

## JSON API

### Users

- `POST /api/users`
- `PATCH /api/users`

### Session

- `POST /api/session`
- `DELETE /api/session`

### Tasks
- `GET /api/tasks`
- `POST /api/tasks`
- `DELETE /api/tasks/:id`

### Taskers
- `GET /api/taskers`
- `GET /api/taskers/:id`

### Reviews
- `GET /api/taskers/:id/reviews`
